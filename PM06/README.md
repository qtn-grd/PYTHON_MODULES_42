## 📜 Module 06 — The Codex

**Mastering Python’s Import Mysteries**

This module covers different import strategies in Python and best practices for writing clean, modular, and maintainable code.

### Concepts Covered

* `__init__.py`
* Absolute vs. relative imports
* Module and package access
* Package metadata
* Aliased imports
* Full module import vs. specific function import
* Late imports
* Circular import issues
