## 🚀 Module 09 - Cosmic Data

**Discover Pydantic Models & Validation**

This module covers Pydantic, a data validation and settings management library. It aims to convert and organize different data types through three space-themed fantasy stories. The last exercise uses nested models to demonstrate examples of complex structures.

### Concepts Covered

* `pydantic`
* `datetime`
* `basemodel`
* `Field`
* `model_validator`
* Nested models
