## 🎮 Module 03 — Data Quest

**Mastering Python Collections for Data Engineering**

This module explores Python’s core data structures, as well as handling command-line arguments. It also introduces generators and comprehensions as more efficient and concise coding patterns.

### Concepts Covered

* Lists
* Dictionaries
* Tuples
* Sets
* Generators
* `yield`
* Time module

### Functions Used

* `sum()`
* `max()`
* `min()`
* `set.union()`
* `set.intersection()`
* `set.difference()`
* `dict.keys()`
* `dict.values()`
* `dict.items()`
* `dict.update()`
* `dict.pop()`
* `iter()`
* `sorted()`
* `dict.get()`
